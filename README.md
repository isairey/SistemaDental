## 🦷 Descripción

Sistema de gestión dental diseñado para clínicas odontológicas y consultorios dentales.  
Permite administrar pacientes, historiales clínicos, tratamientos, citas y procesos administrativos de forma eficiente y organizada.

---

## 🚀 Características Principales

### 👨‍⚕️ Gestión de Pacientes
- Registro de pacientes
- Historial clínico dental
- Odontograma digital
- Seguimiento de tratamientos

### 📅 Gestión de Citas
- Agenda de citas
- Recordatorios de consultas
- Control de disponibilidad

### 🦷 Área Odontológica
- Diagnóstico dental
- Registro de tratamientos (limpieza, extracción, ortodoncia, etc.)
- Control de procedimientos realizados
- Evolución del paciente

### 💊 Tratamientos y Recetas
- Prescripción de medicamentos
- Planes de tratamiento
- Seguimiento clínico

### 💼 Administración
- Gestión de doctores y personal
- Control de pagos
- Reportes administrativos
- Historial de facturación
